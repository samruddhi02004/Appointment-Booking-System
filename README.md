\# VaidyaLink – Appointment Booking ERP



VaidyaLink is a multi-role appointment booking ERP platform designed for healthcare and service-based businesses.



\## Features



\- Resource-based appointment system

\- Customer booking flow

\- Provider dashboard

\- Admin panel

\- Real-time slot management

\- Billing and invoice system

\- Responsive modern UI



\## Tech Stack



\### Frontend

\- Next.js

\- React.js

\- Tailwind CSS



\### Backend

\- Node.js

\- Express.js



\### Database

\- PostgreSQL / MySQL



\## Project Status



🚧 Currently in Development Phase



This project is being developed as part of the Odoo Hackathon.



\## Core Architecture



Provider → Resources → Services → Slots → Appointments
